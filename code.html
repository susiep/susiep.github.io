Agent-based modelling
The GIF above shows several iterations of an agent-based model created using python 3 to model the interactions between an organism, for example a people and its surroundings, for example an orchard . Using this example, to set up the model, the file 'in.txt' (downloadable below) is written in as the environment with a value representing the amount of apples in that location. Then, the number of people (agents), the number of model iterations and the distance in which these people can interact (neighbourhood) can be defined. This step can be run through the command line by writing those three value in that order.
The model then begins the iterations, in which every person completes four functions : moving, XXXXXXX, sharing and XX.XXX. What do these functions do?
